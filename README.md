# career-path-navigator
Career Path Navigator is a Flask-based web application that analyzes user inputs such as interests and skills to recommend relevant career options. The system uses structured logic and data mapping to provide personalized guidance, improving career decision-making.
## Features
- Personalized career suggestions
- Simple UI
- Fast processing

## Tech Stack
- Python
- Flask
- HTML/CSS

## How to Run
1. Install Flask
2. Run app.py
3. Open localhost:5000
